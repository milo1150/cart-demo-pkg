# Tag version

## Create a New Tag

```bash
git tag v1.0.0
```

or

```bash
git tag -a v1.0.0 -m "Release version 1.0.0"
```

## Push the Tag to Remote

```bash
git push origin v1.0.0
```

## Delete a Tag

Delete Locally

```bash
git tag -d v1.0.0
```

Delete on Remote

```bash
git push --delete origin v1.0.0
```
